# 🛍️ UrbanWear Store

A full-stack MERN clothing e-commerce application that provides a modern online shopping experience. Users can browse collections, search products, manage their cart, place orders, and securely authenticate. The project also includes an admin panel for managing products and orders.

---

## ✨ Features

### Customer Features

* User Registration & Login (JWT Authentication)
* Browse Clothing Collections
* Search & Filter Products
* Product Details Page
* Shopping Cart
* Place Orders
* Order History
* Responsive Design

### Admin Features

* Add New Products
* Update Product Details
* Delete Products
* Manage Customer Orders
* Image Upload Support

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/urbanwear-store.git
cd urbanwear-store
```

### Backend Setup

```bash
cd backend
npm install
npm run server
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

CLOUDINARY_NAME=your_cloudinary_name

CLOUDINARY_API_KEY=your_api_key

CLOUDINARY_API_SECRET=your_api_secret
```
